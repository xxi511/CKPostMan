發送卡提諾的短消息      
<img src="https://github.com/xxi511/CKPostMan/blob/master/Img/windows.PNG" width="400">    
*帳密: 填卡提的帳密    
*收件者: 要收短消息的人的帳號，一行一個人     
*訊息: 要發送的訊息    
*進度條: 發送進度    
不能太頻繁發送短消息，每發一條要等30秒，所以發三個人大約要一分鐘      
發送的時候如果看到程式出現【無反應】之類的狀況不要理他，傳好會跳通知
